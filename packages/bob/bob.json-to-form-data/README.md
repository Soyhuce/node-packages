# @soyhuce/bob.json-to-form-data

> JSON to FormData function from [Bob library](../bob).

## Installation

```bash
pnpm add @soyhuce/bob.json-to-form-data
```

## Usage

```ts
import JSONToFormData from '@soyhuce/bob.json-to-form-data'

JSONToFormData({
  testStringKey: 'testValue',
  testBooleanKey: true,
  testNumberKey: 0,
  testNullKey: null,
  testFileKey: file
})
```
